HOW TO ENABLE MOTION PLATFORM SIMULATORS:

1. Copy the corresponding json settings file to the application's data folder:

    Windows:    C:\Users\<user name>\AppData\LocalLow\Perrinn\Project 424
    Mac OS X:   ~/Library/Application Support/Perrinn/Project 424
    Linux:      $XDG_CONFIG_HOME/unity3d or $HOME/.config/unity3d

2. Edit the copied json settings file to specify the host IP to send the motion data to.
